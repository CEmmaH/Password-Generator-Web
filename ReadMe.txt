# Java Web Project Password Generator

## Project Overview

This is a password generator application built using Java Web technologies. 
Users can generate secure random passwords through a web interface. 
The goal of this project is to provide a simple and easy-to-use tool for generating passwords of various complexities.

## Features

- **Generate Random Passwords**: Allows users to define the length and complexity of the password (including uppercase letters, lowercase letters, numbers, and special characters).
- **User Interface**: A simple and intuitive web interface that makes it easy for users to generate passwords.

## Tech Stack

- **Java**: Used for backend logic processing.
- **Servlet**: Handles HTTP requests and responses.
- **JSP**: Used for frontend display.
- **Tomcat**: Serves as the Java Web container to run the application.
- **Maven**: Used for building and managing project dependencies.

## Getting Started

### Prerequisites

- Java 8 or higher
- Apache Tomcat 9 or higher
- Maven 3.6 or higher

### Deploy the project to Tomcat server

- Copy the generated target\password-generator-1.0.war file to the webapps directory of Tomcat.
- start tomecat server
- http://localhost:8080/password-generator-1.0/
